The ``1-my_list`` module
========================

Implementation of ``MyList`` class
----------------------------------

First import ``MyList``` class from the ``1-my_list`` module

		>>> MyList = __import__('1-my_list').MyList

application:

		>>> x = MyList()
		>>> x
		[]

append to MyList:

		>>> x = MyList()
		>>> x.append(5)
		>>> x.append(1)
		>>> x.append(3)
		>>> x
		[5, 1, 3]

print sorted list:

		>>> x = MyList()
		>>> x = [4, 3, 7, 1]
		>>> x.print_sorted()
		[1, 3, 4, 7]
		>>> x
		[4, 3, 7, 1]
